# Storytelling Frameworks

Three story architectures. Every narrative piece chooses one.

## Framework 1 — The Return (Возвращение)
Used for: founder story, student transformation, retreat recap.

**Beats:**
1. The life before — outer success, inner quiet.
2. The crack — a moment that could not be ignored.
3. The search — the path tried.
4. The remembering — the encounter with what was already within.
5. The now — life from the return.

## Framework 2 — The Witness (Свидетель)
Used for: teaching posts, podcast narratives.

**Beats:**
1. A body-level moment observed (someone else, or Victoria's own).
2. The surface interpretation most people would give.
3. The deeper reading the tradition offers.
4. What this means for the listener's life.

## Framework 3 — The Invitation (Приглашение)
Used for: launches, retreat announcements, challenge openings.

**Beats:**
1. Name the longing (in their words).
2. Name the pattern that's keeping it unmet.
3. Offer the return path.
4. Describe what becomes possible.
5. Open the door — without pushing through it.

# Emotional Beat Library

Use these as anchors inside any framework:

| Beat | Language pattern |
|---|---|
| Recognition | «Возможно, вы тоже чувствовали...» |
| Permission | «Вам не нужно быть готовой. Нужно только согласиться прийти.» |
| Tender confrontation | «То, что вы называете усталостью — это тоска по телу.» |
| Sacred memory | «Ваше тело помнит. Оно всегда помнило.» |
| Quiet opening | «И если вы готовы — я оставлю дверь приоткрытой.» |

# Story Ethics
- Never use a student's story without written consent.
- Never compress someone else's 2-year path into a 60-second Reel.
- Always protect vulnerability. Anonymize by default.
