# Inspiration Library

Папка, где Виктория сохраняет ссылки на посты, рилсы, лендинги и другой контент, который её зацепил.

Используется как **референс-библиотека**: когда мы садимся писать новый пост, делать карусель или прогрев — мы заходим сюда и смотрим, что уже резонировало.

---

## Как пользоваться

Минимум усилий при сохранении. Просто скидываешь ссылку в `saved-posts.md` — структуру файла наполню я, когда буду разбирать с тобой.

Самый короткий формат:

```
## YYYY-MM-DD · короткое слово (карусель / рилс / лендинг)
URL: [ссылка]
Что цепляет: [одна строка — твоё ощущение]
```

Большее можно дописать потом — например, в момент, когда мы делаем похожий формат и заходим сюда за идеей.

---

## Что сохранять

| Тип | Зачем |
|---|---|
| **Карусели** | Сборка нарратива через слайды, переходы, типографика |
| **Рилсы** | Хуки в первые 3 секунды, монтаж, звук |
| **Посты с длинным текстом** | Стиль письма, ритм, обрывы |
| **Лендинги / сайты** | Структура страницы, премиум-аэстетика |
| **Email-рассылки** | Темы писем, открытия, призывы |
| **Сториз-форматы** | Стикеры, опросы, движение |
| **Истории брендов** | Founder stories, репозиционирование |

---

## Антипаттерны (НЕ сохраняем)

- «Просветление за 30 дней»
- «Богиня внутри тебя»
- Эзотерический жаргон с чакрами/мантрами
- Громкий маркетинг с эмодзи и капсом
- Аудио-визуальный кринж (даже если виральное)

Сохраняем только то, что соответствует уровню бренда: тихо, точно, премиум.

## Структура папки

```
/content/inspiration/
  README.md                 ← этот файл
  saved-posts.md            ← хронологический лог всех сохранённых ссылок
  _analyzed/                ← (опционально) отдельные файлы для глубокого разбора
                              конкретного поста, когда мы делаем его адаптацию
```
