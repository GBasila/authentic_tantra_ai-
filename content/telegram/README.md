# Telegram-канал — стратегия

## Почему Telegram

Telegram — это **второй главный канал** бренда, не «бэкап Instagram».

| Instagram | Telegram |
|---|---|
| Открытое знакомство | Закрытое присутствие |
| Алгоритм решает кто видит | Все подписчики видят каждый пост |
| Reels, carousels — визуальное | Длинный текст, голосовые — словесное и тихое |
| Аудитория широкая | Аудитория глубокая |
| Tone — «приглашаю» | Tone — «делюсь, как с близким» |
| Реклама / нагрев | Углубление / удержание |

**Здесь живут самые верные подписчики.** Они уже знают, кто Виктория. Они подписались, потому что хотят слышать её на регулярной основе.

## Каденция постов

**3-4 поста в неделю.** Не каждый день. Каждый пост — событие, не «контент».

| День | Время МСК | Тип |
|---|---|---|
| Пн | 9:00 | Размышление / заметка из утренней практики |
| Ср | 18:00 | Поэтическая строка / фрагмент учения |
| Пт | 9:00 | Что-то живое — ответ на вопрос, история, голосовое |
| Вс | (свободно) | По вдохновению — может быть, может не быть |

## Форматы постов

### A. Утренняя заметка (200-400 слов)
Из практики Виктории. Сенсорный момент → одна мысль → тихое завершение.

### B. Поэтическая строка (10-30 слов)
Одна-две фразы. Брошенные как камешек в воду. Без вступления, без объяснения.

### C. Голосовое сообщение (3-7 мин)
Не «контент-голос». Просто Виктория, проходящая, дышащая, говорящая в моменте. Telegram идеален для голосовых.

### D. Длинный пост (500-800 слов)
Раз в 1-2 недели. Углубление какой-то темы. Можно перечитывать.

### E. Картинка + короткая подпись
Фрагмент дневника, окно с утренним светом, чашка чая — что-то простое из её жизни. С одной фразой.

## Что НЕ публикуется в канале

- ❌ Кросс-постинг из Instagram (это разные каналы — разный голос)
- ❌ «Срочные» предложения, лимитированные акции
- ❌ Скриншоты других постов / репосты (за редким исключением — если что-то очень важное)
- ❌ Чужие цитаты в стиле «вдохновляющих»
- ❌ Анонсы новостей мира / политика / комментарии
- ❌ Эмодзи россыпью (исключение: одно 🤍 или 🕊️ изредка)

## Что ДА

- ✅ Реальные моменты из жизни — но всегда с практической / духовной нотой
- ✅ Голосовые — здесь голос работает мощнее текста
- ✅ Поэзия (своя или чужая, с указанием)
- ✅ Фрагменты Каула-текстов в собственном переводе
- ✅ Тихие признания: «сегодня было сложно»
