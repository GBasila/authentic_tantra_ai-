## Как загрузить (пошагово, 20 минут)

### Шаг 1 — AirDrop / sync файлов на iPhone

На Mac:
1. Открой Finder → перейди в `~/alignment-app/authentic_tantra_ai/content/instagram/highlights/`
2. Выдели всю папку `about/` (Cmd+A внутри неё)
3. AirDrop на iPhone → Photos
4. Повтори для `sessions/` и `free/`

После этого у тебя на iPhone в Photos лежат 23 PNG. Они в порядке загрузки (file name = order).

### Шаг 2 — Создание Highlight «обо мне» (5 мин)

1. Открой Instagram на iPhone
2. Профиль → справа вверху «+» → **Highlight** (Актуальное)
3. **Cover image:** выбери `about/cover.png` → обрежет к кругу
4. **Name:** «обо мне» (lowercase, по бренду)
5. **Stories:** добавь все 8 в порядке `story-1.png` → `story-8.png`

⚠️ **Важно:** Instagram требует чтобы stories сначала были **опубликованы как обычные stories**, и только потом ты их добавляешь в highlight. Так что workflow:

1. Сначала опубликуй все 8 как Stories → выбери «Только мои близкие» (если не хочешь чтобы они появились у подписчиков **прямо сейчас**) или «Все» (если ок что подписчики увидят)
2. После 24 часов stories пропадают из ленты, **но остаются в архиве**
3. Профиль → справа вверху «+» → Highlight → выбираешь из архива

**Альтернатива:** просто опубликуй сначала Cover как обложку highlight'а, потом по одной добавляй остальные через «Edit Highlight» → «Add stories»

### Шаг 3 — Создание Highlight «1:1» (5 мин)

Тот же процесс с папкой `sessions/`:
- Cover: `sessions/cover.png`
- Name: «1:1»
- Stories: 7 файлов в порядке

### Шаг 4 — Создание Highlight «дар» (5 мин)

Тот же процесс с папкой `free/`:
- Cover: `free/cover.png`
- Name: «дар»
- Stories: 5 файлов в порядке

### Шаг 5 — Расположение highlights в профиле (1 мин)

После создания всех трёх — Instagram размещает их слева направо в порядке создания.

**Идеальный порядок (слева направо):**
1. **обо мне** (первое что видит новый посетитель)
2. **1:1** (премиум-предложение)
3. **дар** (бесплатная дверь — для тех кто не готов к €400)
4. ... твои travel highlights дальше (Тай, Перу, Bali, Мальдивы, и т.д.)

Чтобы поменять порядок: tap на highlight → правый верх «...» → «Edit» → можно архивировать и пересоздать в нужном порядке (Instagram не даёт прямого drag-to-reorder).

## Ссылки в Stories (link sticker)

На stories CTA (последний в каждом highlight) можно добавить **link sticker** прямо в Instagram:

- **обо мне → story-8:** ссылка на `https://authentic-tantra-ai.vercel.app/`
- **1:1 → story-7:** ссылка на `https://authentic-tantra-ai.vercel.app/sessiya/`
- **дар → story-5:** ссылка на форму LM-подписки на главной (когда настроишь Formspree)

Чтобы добавить link sticker:
1. Открой story (когда публикуешь)
2. Сверху → стикер 😀 → выбери «Link»
3. Введи URL
4. Drag на нужное место
5. Опубликуй

После добавления в highlight — link sticker сохраняется. Посетитель тапает → переходит на сайт.

## Технические детали

- Размер: 1080×1920 (Instagram Story native)
- Формат: PNG (lossless, поддерживает text-rendering без артефактов)
- Размер файла: ~50-150 KB каждый (паper grain + gradients добавляют детали но не тяжело)
- Брендовая палитра: Cream Alba (#F2EADF) · Charcoal (#2B2A28) · Aged Brass (#A68A5B) · Warm Dawn (#F5D9B6)
- Шрифты: Cormorant Garamond Italic / Regular (variable Light 300 — Regular 400)
- Эстетика: matches Post 2 carousel v3 (cinematic editorial)
