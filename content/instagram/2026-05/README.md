## Stories — ежедневно (не в этом файле)

Параллельно с feed-постами, **каждый день 1-2 раза** — живые stories. Не планируются — снимаются на телефон в моменте. Виктория сама. Не делегируется.

Темы stories:
- Утренняя практика (вид из окна, чашка чая, рука на сердце — БЕЗ полного лица, только моменты)
- BTS (за кадром): подготовка к ретриту, поездка к учителю, дорога
- Поэтичная заметка дня (одна фраза на screenshot, фон brand-cream)
- Ответ на вопрос из DM (только если без агрессии)

**Что НЕ публикуется в stories:**
- Полигоны / счётчики / опросы маркетингового типа
- «Свайп вверх — купи»
- Дни рождения, ужины с подругами и т.п. (это другой бренд)
- Реклама партнёров

## Хэштеги — общая стратегия

**Максимум 3 хэштега на пост.** Никаких «массовых» тегов. Только узкие, тематические, контекстные.

### Брендовые (используются всегда)
- `#АутентичнаяТантра`
- `#ПамятьТела`

### Тематические (1 на пост, по содержанию)
- `#ТрадицияКаулы`
- `#ДыханиеПрактика`
- `#ОсознанноеТело`
- `#ВозвращениеВТело`
- `#ЖенскаяГлубина` (только для А1)
- `#ОсознаннаяБлизость` (только для А2)
- `#ТантраБезМистики` (для А3, скептиков)

### Что НЕ использовать
- `#тантра` (слишком засорённый, привлекает не ту аудиторию)
- `#духовность`, `#саморазвитие`, `#осознанность` (общие, не наши люди)
- Английские теги (нам нужна русскоязычная аудитория)
- Теги типа `#любовькоплот` — близко по смыслу, но звучит как нео-тантра
- Любые с цифрами «5шагов», «3секрета» и т.д.

## Что делать если конкретный пост не сработал

Не паниковать. Premium-бренд **не оптимизируется по reach**. Мы смотрим на:

| Метрика | Что значит |
|---|---|
| **Save rate** | >8% — есть глубина |
| **Comment quality** | Длинные, эмоциональные, не «🔥» |
| **DM responses** | Кто пишет лично — самые важные люди |
| **Profile visits** | Растёт ли месяц к месяцу |
| **Followers growth** | Не главное. Главное — кто именно подписывается |

Если пост набрал 200 лайков и 0 эмоциональных комментариев — это не успех.
Если пост набрал 50 лайков и 3 длинных комментария + 5 DM — это победа.
