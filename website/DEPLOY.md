# Деплой сайта Аутентичной Тантры

Три способа запустить сайт. От самого простого к самому правильному.

---

## Способ 1 — Локальный просмотр (1 минута)

Просто чтобы увидеть, как сайт выглядит на твоём компьютере.

1. Открой папку `/Users/vicky/alignment-app/authentic_tantra_ai/website/` в Finder.
2. Двойной клик по `index.html`.
3. Сайт откроется в браузере (Safari/Chrome).

**Что работает локально:**
- Все фото и видео
- Анимации, прокрутка
- Все секции

**Что НЕ работает локально:**
- Email-форма (она пока показывает mailto: fallback — это нормально)
- Аналитика
- Поисковые системы не увидят сайт

---

## Способ 2 — Деплой на Vercel (5 минут) ⭐ РЕКОМЕНДУЕМЫЙ

Бесплатно. Сайт будет доступен в интернете по адресу типа `authentic-tantra.vercel.app`. SSL автоматический. CDN глобальный. Деплой при каждом push в GitHub.

### Шаги:

1. **Зайди на [vercel.com](https://vercel.com)** и зарегистрируйся через GitHub (используй тот же аккаунт `GBasila`).

2. **На главной странице** нажми **«Add New… → Project»**.

3. **Выбери репозиторий** `GBasila/authentic_tantra_ai-` из списка → **Import**.

4. **На странице настройки** укажи:
   - **Framework Preset:** `Other` (это статический сайт без фреймворка)
   - **Root Directory:** нажми **Edit** → выбери `website` (важно!)
   - **Build Command:** оставь пустым
   - **Output Directory:** оставь пустым
   - **Install Command:** оставь пустым

5. **Нажми Deploy**. Через 30–60 секунд получишь ссылку: `authentic-tantra-xxx.vercel.app`.

6. **Открой ссылку** → проверь, что всё работает.

## Способ 3 — Подключение своего домена (10 минут)

После того как деплой работает на vercel.app, можно подключить **свой домен**.

### Если у тебя ещё нет домена:

Покупай на **[namecheap.com](https://namecheap.com)** (€10–15/год). Хорошие варианты:
- `authentictantra.com`
- `authentic-tantra.com`
- `victoria-tantra.com`

Я бы взяла `.com` версию. `.ru` сейчас сложнее для международных платежей.

### Подключение домена к Vercel:

1. В **Vercel** → твой проект → **Settings → Domains**.
2. Нажми **Add Domain** → введи `authentictantra.com`.
3. Vercel покажет, какие **DNS-записи** нужно добавить у регистратора.
4. Зайди в **Namecheap → Domain List → Manage → Advanced DNS**.
5. Добавь две записи (как покажет Vercel):
   - Тип `A`, host `@`, value `76.76.21.21`
   - Тип `CNAME`, host `www`, value `cname.vercel-dns.com`
6. Подожди 5–30 минут.
7. Проверь: открой `authentictantra.com` — увидишь свой сайт.

SSL-сертификат Vercel выдаст автоматически (HTTPS заработает через несколько минут после подключения домена).

## После деплоя — что улучшить

### Email-форма (ВАЖНО)

Сейчас в `index.html` стоит плейсхолдер для **Formspree** (бесплатный сервис форм для статических сайтов).

**Чтобы email-форма реально работала:**

1. Зайди на [formspree.io](https://formspree.io) → зарегистрируйся (бесплатно — до 50 писем/мес).
2. Создай новую форму → дай ей название «5 дней возвращения».
3. Скопируй URL формы (что-то вроде `https://formspree.io/f/xqkrjwgz`).
4. Открой `website/index.html` в текстовом редакторе.
5. Найди строку:
   ```html
   <form class="magnet__form" action="https://formspree.io/f/REPLACE_WITH_YOUR_FORM_ID"
   ```
6. Замени `REPLACE_WITH_YOUR_FORM_ID` на свой ID (например `xqkrjwgz`).
7. Закоммить и запушь — Vercel сразу пере-деплоит.

**Альтернатива** (если позже захочешь): подключить ConvertKit / Kit / Mailerlite напрямую — там тоже есть HTML embed форм.

### Реальная ссылка на Telegram

В футере есть ссылка на `https://t.me/authentic_tantra`. Если канал назван иначе — поменяй в `index.html`.

### Реальная ссылка на Instagram

То же самое — `https://instagram.com/authentic_tantra`. Подставь свой реальный handle.

### Подкаст — пока заглушка

В футере ссылка на подкаст ведёт на `#`. Когда подкаст будет — заменим на реальную ссылку (Apple Podcasts / Spotify).

### Favicon (значок в табе браузера)

Сейчас табы покажут просто текст. Чтобы появился значок «А» (твой инициал):

1. Сделай файл `favicon.png` (512×512 px, серифом «А» на cream фоне).
2. Положи в `/website/`.
3. Добавь в `<head>` строку:
   ```html
   <link rel="icon" type="image/png" href="/favicon.png" />
   ```

## Если что-то сломается

Самые частые проблемы:

| Проблема | Причина | Решение |
|---|---|---|
| Битая иконка вместо фото | Файл не сохранён или другое имя | Проверь `PHOTOS_NEEDED.md` — точные имена |
| Видео не играет | Браузер блокирует autoplay | Постер всё равно показывается — это норма |
| Шрифты «прыгают» при загрузке | Google Fonts грузятся | Это норма для первого визита |
| Сайт долго грузится | Видео большое | Сейчас 13MB — можно сжать сильнее |

Если что-то посерьёзнее — пиши мне, разберёмся.
