## ⚙️ Технические рекомендации

Чтобы сайт грузился быстро и красиво:

1. **Размер**: 1200–1600 px по большей стороне (больше не нужно — сайт всё равно отрендерит меньше)
2. **Формат**: JPG, качество 80–85% (или WebP если знаешь как)
3. **Цветовой профиль**: sRGB
4. **Имена файлов**: строго как выше, **lowercase**, латиница, дефисы (не нижние подчёркивания)

### Как уменьшить фото быстро

**На Mac**: открой фото в Preview → File → Export → JPEG, качество 80%, можно задать размер.

**Онлайн**: [squoosh.app](https://squoosh.app) — перетащи фото, выбери JPG, качество 80, скачай.
