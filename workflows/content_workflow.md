# Daily Content Workflow

## Sunday — Capture
- 14:00-16:00 · Victoria + Content Manager
- 2-hour recording session: voice memos, short videos, photo set
- Output: 10 short videos, 30 voice notes, 5 photo sets
- Assets deposited in shared drive → linked into `/content/raw/YYYY-MM-DD/`

## Monday — Listen
- 08:00 · Audience Agent runs weekly cycle
- 10:00 · Audience dashboard delivered to team
- 11:00 · Content Manager reviews dashboard + selects week's themes from `/content/content_calendar.md`

## Tuesday — Draft
- 09:00 · Content Agent drafts week's content (3 variants per slot)
- 14:00 · Copywriter refines selected variants

## Wednesday — Audit
- 10:00 · Brand Guardian Agent audits all drafts
- 13:00 · Victoria 30-min voice review on anything flagged

## Thursday — Schedule
- 09:00 · Content Manager loads approved content into Buffer/Later
- 12:00 · Visual assets finalized by Creative Director (for carousels, hero images)
- 17:00 · Weekly content locked

## Daily — Engage
- 09:00, 13:00, 19:00 · Community Manager runs comment engagement windows
- Throughout: Victoria 1x/day records raw story live on phone (not scheduled)

## Friday — Reflect
- 16:00 · Mini retrospective: what resonated, what didn't, what to adjust

---

## Handoffs
1. Victoria → Content Manager (raw capture)
2. Audience Agent → Content Agent (signals)
3. Content Agent → Copywriter (drafts)
4. Copywriter → Brand Guardian (refined)
5. Brand Guardian → Victoria (if flagged)
6. Victoria → Content Manager (approved)
7. Content Manager → channels (published)
8. Content Manager → `/content/archive/YYYY-MM/` (archived)

## Approval gates
- Brand Guardian PUBLISH verdict required before any merge to `main` in `/content/`
- Victoria's voice review required if flagged
- Community Manager approves engagement tone weekly
