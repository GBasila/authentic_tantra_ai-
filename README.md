# Аутентичная Тантра · AI Operating System

> «Тут нечему учиться — здесь есть лишь то, что нужно вспомнить.»

This repository is the internal operating system of the Authentic Tantra brand —
a premium transformational space rooted in the Kaula tantric tradition,
led by Victoria. It is a living system, not a static archive.

## Purpose
To protect the soul of the brand, the founder's voice, the premium positioning,
and the client experience — while making the business scalable and organized.

## Architecture
- `/brand/` — the immutable soul (voice, identity, visual, mission)
- `/audience/` — the listening (who we serve, what they carry)
- `/content/` — the living voice (daily creative output)
- `/sales/` — the invitation (offers, launches, scripts)
- `/retreats/` — the sacred container (retreat design, journey)
- `/strategy/` — the horizon (plans, roadmap, resources)
- `/agents/` — the AI workforce (six specialized agents)
- `/templates/` — proven patterns (reusable starting points)
- `/workflows/` — how the system breathes (orchestration)

## For contributors
1. Start with `/brand/brand_identity.md` and `/agents/_shared_context.md`.
2. Read `CONTRIBUTING.md` for commit conventions and collaboration rituals.
3. Stay in your folder. If your work touches `/brand/`, open a PR tagged for Victoria.
