# Launch · Аутентичная Глубина · Поток 1

Первый онлайн-поток флагманского курса (Tier 2 в Offer Ladder). Это не новый продукт — это реализация уже описанной стратегии в `/sales/offers.md`.

## Ключевые параметры

- **Формат:** 8 недель, групповой онлайн
- **Размер группы:** 12–15 женщин
- **Состав:** еженедельный живой созвон в Zoom (90 мин) + закрытый Telegram-чат + ежедневная практика 15–20 мин + 1 индивидуальная сессия 30 мин с Викторией
- **Цена:** €450 (доступная для первого онлайн-потока, ниже флагманского диапазона €600–900 — намеренно, чтобы собрать первую когорту)
- **Дата старта:** через 4 недели после возвращения из 10-дневного поиска видения
- **Цель потока:** €5 400 (12 учениц) — €6 750 (15 учениц)

## Приоритет

Деньги. Маленькая глубокая группа > большая поверхностная. Лучше 12 платящих по €450, чем 30 по €100.

## Главный канал прогрева

Личные приглашения тёплой базе. Это главный актив — женщины, которые уже были на офлайн-группах и VIP за 2 года. С маленькой подписной базой (<500) контентный охват не сработает; работает прямой контакт.

## Поддерживающие каналы

- Один открытый бесплатный мастер-класс на 90 мин (главная точка конверсии)
- Прогрев в Instagram + Telegram-канале: 14 дней постов и сторис
- Финальное письмо/пост о закрытии набора с дедлайном

## Воронка движения

1. База: 50–80 тёплых контактов
2. Личное сообщение: 50–80 отправок
3. Отклик: 25–40 ответов
4. Регистрация на мастер-класс: 20–30
5. Пришли на эфир: 12–20
6. Оплатили курс: 10–15
